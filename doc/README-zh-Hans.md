# Easy I18n

<img src="../res/icon_256x256.png" width=150>

**該軟體用於解決軟體開發和營運過程中多語系管理的難題。**

該專案基於 Kotlin Multiplatform 開發，支援 MacOS、Windows 以及 Linux。

## 功能

在過去，當我們維護多語系詞條時，對一個詞條的修改可能需要查找多個多語系詞條文件。Easy I18n 巧妙地將這些詞條的編輯整合在一個對話框內完成，操作更加便捷。主要功能：

- 載入與顯示多語系詞條，直觀呈現、快速操作
- 支援刪除、修改詞條，並寫回多語系文件
- 支援自動翻譯
- 支援多種開發平台：Android、iOS、Flutter、Compose 以及 Java 等
- 支援關鍵字查詢與多語系篩選等
- 其他功能…

該專案已開源，歡迎你為專案貢獻程式碼或提出建議～ 🎉

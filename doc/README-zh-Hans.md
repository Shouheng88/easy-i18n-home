## 功能

- 載入並展示多語言詞條，介面直觀，操作便捷
- 支援刪除、修改詞條，並將變更回寫至多語言檔案
- 支援多種翻譯模式：Google、DeepL、騰訊、百度、Libre 等
- 支援 AI 翻譯，可新增備註資訊提升翻譯精準度
- 支援多種開發平台：Android、iOS、Flutter、Compose 以及 Java 等
- 支援關鍵字查詢與多語言詞條過濾

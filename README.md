## 功能

- 支持多种翻译模式：Google、DeepL、Tencent、Baidu、Libre 等
- 支持 AI 翻译，允许添加备注信息以使翻译更加准确
- 支持多种开发平台：Android、iOS、Flutter、Compose 以及 Java 等
- 支持查询关键字和过滤多语言等

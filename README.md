## Features

- Load and display multilingual entries with an intuitive interface and quick operation capabilities.
- Support deleting and modifying entries, with changes written back to multilingual files automatically.
- Support multiple translation modes: Google Translate, DeepL, Tencent Translate, Baidu Translate, Libre Translate, etc.
- Support AI-powered translation, allowing users to add notes for more accurate translation results.
- Support multiple development platforms: Android, iOS, Flutter, Compose, Java, etc.
- Support keyword search and multilingual entry filtering.
